## Usage

These examples are provided for reference purposes. They demonstrate:

1. How to execute the system
2. What output to expect
3. How to interpret results

## Sample Execution

See [sample_execution.md](sample_execution.md) for a complete example of running the system from start to finish.

## Expected Output

See [expected_output.md](expected_output.md) for examples of expected output files and their formats.

## Creating Your Own Examples

To create your own examples:

1. Copy the sample data from `sandbox/sample_data/`
2. Run the system using the unified launcher
3. Document the steps and results
4. Save in this directory with descriptive filename

## Best Practices

- Keep examples simple and clear
- Include expected output
- Document any assumptions
- Test examples before sharing
